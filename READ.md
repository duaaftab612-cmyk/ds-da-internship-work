# 📊 Task 1: Term Deposit Subscription Prediction (Bank Marketing)

---

## 🎯 Objective
Predict whether a bank customer will subscribe to a term deposit based on marketing campaign data.

---

## 📂 Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

---

## 🛠 Approach

### 1️⃣ Data Exploration
- Loaded dataset and checked missing values
- Explored categorical and numerical features

### 2️⃣ Data Preprocessing
- Encoded categorical variables
- Handled imbalanced data issues

### 3️⃣ Model Building
- Logistic Regression
- Random Forest Classifier

### 4️⃣ Model Evaluation
- Confusion Matrix
- F1 Score
- ROC Curve

### 5️⃣ Explainability
- Used SHAP/LIME to explain predictions for sample customers

---

## 📈 Results
- Random Forest performed better than Logistic Regression
- Key features: duration, campaign, previous outcome

---

## 💻 Tools Used
- Python
- Pandas
- Scikit-learn
- SHAP / LIME

---

## 🚀 Conclusion
This project helps understand customer behavior and improve targeted marketing strategies.

👥 Task 2: Customer Segmentation Using Unsupervised Learning

---

## 🎯 Objective
Cluster customers based on spending behavior and identify marketing strategies for each segment.

---

## 📂 Dataset
Mall Customers Dataset

---

## 🛠 Approach

### 1️⃣ Exploratory Data Analysis (EDA)
- Analyzed age, income, and spending score
- Visualized distributions and relationships

### 2️⃣ Data Preprocessing
- Selected relevant features for clustering
- Scaled data for better performance

### 3️⃣ Clustering Model
- Applied K-Means Clustering
- Found optimal clusters using Elbow Method

### 4️⃣ Visualization
- Used PCA / t-SNE for cluster visualization

---

## 📊 Insights
- High-income high-spending customers identified
- Low-income low-spending group found
- Distinct customer segments discovered

---

## 🎯 Business Strategy Suggestions
- Premium offers for high-value customers
- Discounts for low-spending segments
- Targeted marketing campaigns

---

## 💻 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

---

## 🚀 Conclusion
This project demonstrates how clustering helps in customer segmentation and business decision-making.
