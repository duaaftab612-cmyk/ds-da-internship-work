## 📊 Task 1: Term Deposit Subscription Prediction (Bank Marketing)

### 🎯 Objective
Predict whether a customer will subscribe to a term deposit based on marketing campaign data.

### 🛠 Techniques Used
- Data Cleaning & Preprocessing
- Label Encoding / One-Hot Encoding
- Logistic Regression
- Random Forest Classifier
- Model Evaluation (F1 Score, Confusion Matrix, ROC Curve)
- Explainable AI (SHAP / LIME)

### 📌 Outcome
Built a classification model to understand customer behavior and improve marketing targeting.

---

## 👥 Task 2: Customer Segmentation Using K-Means Clustering

### 🎯 Objective
Segment customers based on spending behavior and income patterns.

### 🛠 Techniques Used
- Exploratory Data Analysis (EDA)
- Feature Scaling
- K-Means Clustering
- Elbow Method
- PCA / t-SNE Visualization

### 📌 Outcome
Identified distinct customer groups for targeted marketing strategies.
